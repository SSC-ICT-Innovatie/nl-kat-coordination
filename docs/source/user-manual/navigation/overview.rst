Overview of pages
=================

This manual covers the day-to-day use of OpenKAT in the user interface. If you are looking for how to navigate around in OpenKAT, you are at the right place.

The user interface of OpenKAT consists of the following main navigation pages, which provide access to various functionality and data:

- **Organizations:** The Organizations page provides an overview of all organizations you have access to. Select an organization to view its security data and configure settings.
- **Plugins:** The Plugins page shows all available scanning plugins and their configurations. Plugins automatically run when objects meet the required clearance level.
- **Findings:** The Findings page gives an overview of all security findings discovered by plugins and business rules.
- **Objects:** The Objects page contains an overview of all the objects that are added manually or discovered through scanning.
- **Tasks:** The Tasks page contains an overview of all tasks and their status (e.g., completed and failed). You can view task details and download raw output files here.
- **Schedules:** The Schedules page allows you to create and manage recurring tasks that run automatically on a schedule.
- **Files:** The Files page contains all files uploaded to or generated by OpenKAT, including raw scan outputs and report PDFs.
- **Business rules:** The Business rules page allows you to create custom SQL queries that generate findings based on your specific security policies.
- **Reports:** The Reports page allows you to generate and view security reports in PDF format.

Additional pages accessible from the secondary menu:

- **Members:** The Members page (accessible via organization settings) contains an overview of all users that have access to the selected organization, including their roles and clearance levels.
- **Settings:** The Settings page (accessible via organization settings) allows you to configure organization settings and scanning indemnification.
- **User settings:** Clicking in the top right corner on your account allows you to access 'My Organizations', 'Profile' settings, and session logout.

Each of the above-mentioned pages will go into depth on the functionality and working of that page.
